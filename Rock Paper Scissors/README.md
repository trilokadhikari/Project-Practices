# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## Features
- Play against a computer opponent
- Choose Rock, Paper, or Scissors with on-screen buttons
- See the computer's choice and the round result
- Keep track of player and computer scores
- Visual feedback for wins, losses, and ties

## Project Files
- index.html - Main structure of the game interface
- style.css - Styling for the layout and visuals
- script.js - Game logic and score tracking

## How to Play
Click one of the emoji buttons to make your move. The game will compare your choice with the computer's random selection and display the result.
