# Clock App

A modern, interactive clock application built with React and Vite. This project provides three essential time-related utilities: a live digital watch, a stopwatch for timing activities, and a countdown timer.

## Features

- **Watch** - Real-time digital clock displaying the current time
- **Stopwatch** - Start, stop, and reset timer for measuring elapsed time
- **Timer** - Countdown timer for setting specific time intervals
- Clean, responsive UI built with Tailwind CSS

## Tech Stack

- **React 19** - UI library for building interactive components
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework for styling
- **Oxlint** - Fast JavaScript/TypeScript linter

## Components

### Watch
Displays the current time in real-time, automatically updating every second.

### Stopwatch
Allows users to:
- Start and stop the timer
- Reset the timer to zero
- Display elapsed time in hours, minutes, seconds, and milliseconds

### Timer
Provides a countdown timer where users can:
- Set a desired time duration
- Start the countdown
- Pause or reset the timer
