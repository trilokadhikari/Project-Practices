# Dice Roller with React

A simple dice rolling app built with React, Vite, and Tailwind CSS. Input the number of dice you want to roll and click the button to generate random dice results and see matching dice face images.

## Features

- Select how many dice to roll
- Random dice values between 1 and 6
- Display numeric results and matching dice face images
- Built with React and styled using Tailwind CSS

## Demo

The app renders a minimal interface with:

- a numeric input for the number of dice
- a `Roll` button
- a result list of dice values
- dice face images that update after each roll


## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4

## Notes

- The dice results are generated randomly using `Math.random()`.
- The number input is constrained to a minimum value of `1`.
