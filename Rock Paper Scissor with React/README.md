# Rock Paper Scissors with React

A simple Rock-Paper-Scissors game built with React, Vite, and Tailwind CSS.

## Features

- Interactive game controls for Rock, Paper, and Scissors
- Random computer opponent selection
- Immediate win/lose/tie result display
- Live score tracking for player and computer
- Clean UI using Tailwind CSS classes


## Notes

- The game uses `useState` and `useEffect` for state management and score updates.
- The current app is designed for simple desktop/mobile interaction with emoji buttons.
