## Password Generator

A simple React + Vite password generator built with Tailwind CSS. Generate secure passwords with adjustable length, optional numbers, and optional special characters, then copy the result to the clipboard.

---

### Features

- Auto-generates a new password
- Adjustable password length (6 to 100)
- Option to include numbers
- Option to include special characters
- One-click copy to clipboard
- Styled with Tailwind CSS
- Built with React and Vite

---

### Demo

> Enter a length, toggle Numbers and Characters, then copy the generated password.
