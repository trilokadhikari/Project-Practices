# Temperature Conversion

A simple browser-based temperature converter that lets users convert between Celsius and Fahrenheit.

## Demo

Open `index.html` in a web browser, enter a temperature value, choose a conversion direction, and click **Submit**.

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Displays results with one decimal place
- Provides validation when no conversion option is selected

## Files

- `index.html` � page structure and form controls
- `style.css` � styling for form layout and interactive button
- `script.js` � conversion logic and result display

## How it works

1. Enter a numeric temperature in the input box.
2. Choose either **Celsius to Fahrenheit** or **Fahrenheit to Celsius**.
3. Click **Submit**.
4. The converted value appears below the button.

## Conversion formulas

- Celsius to Fahrenheit: `F = C � 9/5 + 32`
- Fahrenheit to Celsius: `C = (F - 32) � 5/9`
