# Currency Converter

A modern and simple currency converter built with React, Vite, and Tailwind CSS. This project allows users to convert amounts between different currencies with a clean and responsive interface.

## Features

- Convert amounts between multiple currencies
- Swap source and target currencies with one click
- Responsive UI for desktop and mobile devices
- Built with a modern React + Vite setup

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript
